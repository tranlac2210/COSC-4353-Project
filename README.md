# Installation Instruction:

**Front-end:**

**Step 1:** Install dependencies
```
npm install
```

**Step 2:** Start the development server
``` 
npm run start
```

**Back-end:**

**Step 1:** Install dependencies
```
npm install
```

**Step 2:** Start the development server
``` 
npm run dev
```

**Unit test:**

**Step 1:** Install dependencies
```
npm install
```

**Step 2:** Start the development server
``` 
npm run dev
```
